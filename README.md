当你是一个大一新生 加入了实验室 你的导师让你去参加那个智能车竞赛
你想我马了个蛋了刚看完江协（其实没看完
就来搞这什么鬼龙芯 编译不能直接编译 还要装虚拟机 配网 还要搞Cmake这什么鬼
最难的是你还要在这上面搞一个什么鬼循迹我真是日了
然后你思索一阵 打开Bilibili ，你发现了上交大八领域这六个字
https://www.bilibili.com/video/BV1VN411N79B，他说绝对可以跑，还开源代码
你根本看不懂C++什么枚举、类啥的 所以你点开百度网盘先把资源下载下来
你发现 搞这个可能需要丰富的经验和知识 开始忽悠队友先搞pid
你看着代码 整不会了 打开了TraeAI之类的IDE 全部丢给AI问这玩意是啥
AI给你讲了一通 你听不懂 你说：别说了直接帮我整理成一个可以运行的代码吧
AI费尽心血（tokens）帮你写完，配好环境，找到opencv，然后告诉你找不到摄像头
你发现这龙芯真的是世界上最难搞的东西
你决定先从你熟悉（并非）的Python开始弄起
Python的OpenCV库确实好弄 两行代码就把摄像头给跑起来了
你很高兴 因为起码屏幕里已经有二值化的图像和一点点红色边线了
.......
一两个月之后 在逐飞龙邱的群里混迹许久 你已经在屏幕中看到了偏移值、帧率等高阶信息
你非常自豪 端着电脑拿给队友看 并决定是时候转译到C++了

然后你就遇到了世界上最水深火热的坑
